h3. 'The People Reviewer'

- An application that lets users review other people. 
- It integrates with Facebook Authentication and Graph search, requiring users already have a Facebook profile to interact with the site as well as to be reviewed, to both limit duplication of reviewees as well as validate their actual existence. 
- Anonymous commenting function originally existed, but has been commented out; still viewable within the code. 
- Built using Ruby on Rails, HTML, CSS, OAuth, Facebook API, Graph Search API
